# STRIDALGO - Features Roadmap

## Version 1 (MVP)

### Authentication System
**Purpose:** Allow users to create and manage accounts.

**User Actions:**
- Sign up
- Login
- Logout
- Reset Password

**System Actions:**
- Store user information
- Hash passwords
- Generate authentication tokens
- Protect private routes

---

### Dashboard
**Purpose:** Display user progress.

**User Actions:**
- View XP
- View coins
- View floors
- View streaks

**System Actions:**
- Fetch user statistics
- Calculate progress
- Display recent activity

---

### Question Engine
**Purpose:** Allow users to solve DSA questions.

**User Actions:**
- Select difficulty
- Solve questions
- Submit code

**System Actions:**
- Fetch questions
- Execute code
- Evaluate solutions
- Store submissions

---

### Reward Engine
**Purpose:** Reward users for solving problems.

**User Actions:**
- Earn XP
- Earn coins
- Earn medals

**System Actions:**
- Calculate rewards
- Update statistics
- Increase building floors

---

### Basic City Builder
**Purpose:** Visualize user progress.

**User Actions:**
- View city
- Watch buildings grow

**System Actions:**
- Add floors
- Save city state
- Update skyline

---

### Leaderboard
**Purpose:** Create competition among users.

**User Actions:**
- View rankings
- Compare progress

**System Actions:**
- Calculate rankings
- Update scores
